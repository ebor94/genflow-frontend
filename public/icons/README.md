# Iconos PWA

Estos archivos son placeholders y deben reemplazarse antes del go-live por iconos definitivos del cliente Serfunorte (logo corporativo).

Tamaños requeridos:
- `icon-192.png` — 192×192px, fondo cream (#F6F1E9), logo brown-deep (#2C1A0E)
- `icon-512.png` — 512×512px, mismo logo
- `icon-maskable.png` — 512×512px con safe area de 80px (margen para "adaptive icons" Android)
- `apple-touch-icon.png` — 180×180px (iOS home screen)

Mientras tanto se generan PNG mínimos válidos vía script `generate-placeholders.js`.
